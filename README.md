# ST Launcher

ST Launcher 是一个基于 **Tauri 2 + Leptos** 开发的 SillyTavern 桌面启动器，目标是提供一个开箱即用、便于维护的本地启动环境。

本项目用于辅助安装、更新和启动 SillyTavern，并尽量将运行依赖、终端输出和相关数据控制在本地应用目录内，避免污染系统环境。

## 项目特性

- 桌面端应用：基于 Tauri 2 与 Rust 后端构建。
- 前端界面：使用 Leptos + Trunk 构建。
- 本地运行时：支持便携式 Node.js 与 Git 的检测、下载和管理。
- SillyTavern 管理：支持安装、更新、启动与停止。
- 内置终端：用于查看安装、更新、启动等过程中的输出信息。
- 本地优先：依赖和数据尽量保存在应用目录内。

## 技术栈

| 模块 | 技术 |
| --- | --- |
| 桌面框架 | Tauri 2 |
| 后端 | Rust |
| 前端 | Leptos |
| 构建 | Trunk / Cargo |
| 打包 | Tauri Bundle |

## 运行方式

请先确保本机已经安装 Rust、Cargo、Tauri CLI 与 Trunk。

在项目根目录执行：

```powershell
cargo tauri dev
```

项目的 `tauri.conf.json` 已配置 `beforeDevCommand: trunk serve`，运行上述命令后会自动启动前端并打开桌面窗口。

如果提示找不到 `trunk`，请先安装：

```powershell
cargo install trunk
```

## npm 脚本

本项目不是 Node/React 项目，根目录中的 `package.json` 仅用于转发 Cargo/Tauri 命令。

```powershell
npm run dev
npm run build
npm run tauri -- dev
```

通常不需要为了运行应用而执行 `npm install`。

## 前端单独预览

如只需要预览前端界面，可以执行：

```powershell
trunk serve
```

然后在浏览器访问：

```text
http://localhost:1420
```
